# SN66 Miner Agent

Bittensor Subnet 66 mining agent. Based on the [tau agent format](https://github.com/unarbos/tau).

## Setup

1. Register hotkey on subnet 66
2. Push this repo to GitHub
3. Run `commit_agent.py` to commit your repo reference to the chain

## Commitment

```bash
pip install bittensor
python commit_agent.py
```
